🔍 方法概述

我们的方法利用统一的 3D 高斯表示，联合建模运动模糊和散焦模糊，跨多视角图像进行重建。关键贡献包括：

  • 引入双重模糊感知模块，生成像素级模糊掩码，显式减轻运动和散焦模糊对 3D 重建的影响。
  • 设计模糊感知的高斯溅射过程，结合模糊掩码进行梯度优化，确保准确建模，避免信息退化。
  • 提出 UGR-Opacity 自适应优化，在欠优化的模糊区域细化高斯表示，提高准确性和质量。
